## Use

### Deno

Add [the library from JSR](https://jsr.io/@lambdaurora/lambdawebserver):

```shell
deno add @lambdaurora/lambdawebserver
```

Then import it:

```typescript
import { /* Whatever you need. */ }from "@lambdaurora/lambdawebserver";
```
